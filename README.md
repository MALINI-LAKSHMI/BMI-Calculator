# BMI-Calculator
This project is a BMI (Body Mass Index) Calculator built using React.
It allows users to enter their height and weight and instantly calculates their BMI value along with the corresponding health category.
The application is simple, responsive, and easy to use.

## Technologies Used

React

JavaScript

HTML

CSS
